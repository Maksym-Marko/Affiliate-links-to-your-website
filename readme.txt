== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Affiliate Links WooCommerce'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Affiliate Links WooCommerce'.
2. Upload the 'Affiliate Links WooCommerce' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Affiliate Links WooCommerce' from your Plugins page.
